# assets/fonts

## Citadel Script (fonte do script — nomes, títulos, assinatura)

**Citadel Script** não está no Google Fonts (é uma fonte premium, gratuita para uso pessoal).
Para ativá-la no site, baixe o arquivo da fonte e salve **aqui** com um destes nomes:

```
assets/fonts/citadel-script.woff2   (melhor — menor/mais rápido)
assets/fonts/citadel-script.otf
assets/fonts/citadel-script.ttf
```

O `@font-face` já está configurado em `css/tokens.css` apontando para esses nomes.

Enquanto o arquivo não estiver aqui, o site usa **Pinyon Script** (do Google Fonts) como
fallback — bem parecida com a Citadel, então nada quebra.

> Dica: se você tiver só o `.otf`/`.ttf`, dá pra converter para `.woff2` em conversores online
> (ex.: cloudconvert / transfonter) para o site carregar mais rápido.

As outras duas fontes (**Cormorant Garamond** e **Merriweather**) vêm do Google Fonts, já
carregadas no `index.html` — não precisa fazer nada.
